树莓派操作系统大全（史上最全）
支持树莓派的操作系统
目录

转载自科技爱好者博客 (opens new window)并添加了更多内容。

树莓派的系统镜像非常多，绝大部分Linux发行版都发布了支持树莓派的操作系统，还有许多小众的系统。今天我再次归纳了树莓派的操作系统镜像，可以说是史上最全的树莓派操作系统镜像。

  • 9front (opens new window) - 贝尔实验室 Plan 9 的分支，支持一些新的软硬件
  • Alpine Linux (opens new window)-基于musl libc和busybox的面向安全的轻量级Linux发行版。
  • Android Things (opens new window)-构建适用于各种消费，零售和工业应用的连接设备。
  • Arch Linux ARM (opens new window)- 轻巧灵活的Linux发行版，力求使其简单。
  • balenaOS (opens new window)- 用于在嵌入式设备上运行Docker容器的开源OS，其设计旨在提高可靠性并在生产中得到证明。
  • BerryBoot (opens new window) - Bootloader /通用操作系统安装程序，支持VNC和HDMI-CEC。
  • CentOS (opens new window) - 树莓派上的CentOS。
  • chilipie-kiosk (opens new window)- 直接引导至全屏Chrome的镜像，非常适合仪表板和显示器。
  • Chromium OS (opens new window)- Chromium OS for Raspberry Pi
  • Debian-Pi-Aarch64 (opens new window)- 树莓派爱好者基地 64 位系统
  • DietPi (opens new window)- 设计成可容纳2GB SD卡的最小图像，具有大量可配置的设置和脚本。
  • DroneBridge (opens new window) - WifiBroadcast扩展，可替代DJI Lightbridge和其他类似系统。
  • ESXi (opens new window) - 具有高级资源管理功能，高效，灵活的虚拟机平台
  • EZ-WifiBroadcast (opens new window)- 轻松又廉价的无线数字高清视频传输。
  • FabScanPi (opens new window) - FabScanPi是使用Raspberry Pi摄像头模块的开源3D激光扫描仪。
  • Fedora (opens new window)- 为树莓派构建的Linux Fedora发行版。
  • FreeBSD (opens new window) - FreeBSD是一种高级计算机操作系统，用于为现代服务器、台式机和嵌入式平台提供动力。
  • Gentoo (opens new window) - 树莓派的Gentoo Stage 3压缩包。
  • Gladys Assistant (opens new window) - Gladys，隐私优先的开源家庭助理。
  • Hass.io (opens new window)-嵌入式设备的家庭自动化操作系统/应用程序，也可以独立使用。
  • HypriotOS (opens new window)- 最小的基于Debian的操作系统，经过优化可运行Docker。
  • IchigoJam RPi (opens new window)- 将Raspberry Pi直接引导到IchigoJam BASIC
  • Kali Linux (opens new window)- 用于ARM设备的渗透测试和道德黑客Linux发行版。
  • Kano OS (opens new window)-用于探索，创建和玩耍的开源OS，系统免费。
  • Lakka (opens new window) -树莓派复古游戏镜像，完全以RetroArch为基础开发。
  • LibreELEC (opens new window)- Kodi的操作系统
  • LineageOS (opens new window) - 在树莓派上运行 Android 10
  • Linutop OS (opens new window) - 一个安全的基于 Raspbian 的网络信息亭和数字标牌播放器
  • Lubuntu (opens new window) - 以 LXDE 为特色的发行版
  • Manjaro (opens new window) - 树莓派的友好、开源Linux发行版。
  • Minibian (opens new window)- 最小的Raspbian镜像（比Jessie Lite还小）。
  • moOde (opens new window) - 出色的树莓派发烧级质量音乐播放器
  • MoodleBox (opens new window) - MoodleBox在树莓派上提供了Moodle学习管理系统。
  • motionEyeOS (opens new window) - Linux发行版，可将单板计算机转变为视频监视系统。
  • Mozilla WebThings (opens new window)- 建立自己的智能家居系统
  • NetBSD (opens new window) - NetBSD是一个免费、快速、安全且高度可移植的类Unix开源操作系统。
  • NextCloudPi (opens new window)-基于Raspbian的Nextcloud镜像。具有在Raspbian 8上运行的Nextcloud 11，具有启用PHP 7和HTTP2的Apache服务器。
  • NOOBS (opens new window)-全新的开箱即用软件，为初学者提供了一个简单的OS安装程序。
  • OctoPi (opens new window)-用于3d打印机的发行版。
  • OpenBSD (opens new window)- 免费的类 Unix 操作系统
  • OpenELEC (opens new window)- 开放式嵌入式Linux娱乐中心，一种非常轻便的Kodi版本。
  • openEuler (opens new window) - 华为服务器操作系统 EulerOS 的开源版本
  • OpenHABian (opens new window) - OpenHAB家庭自动化软件的预配置版本。
  • OpenMediaVault (opens new window) -OpenMediaVault是基于Debian Linux的下一代网络连接存储（NAS）解决方案，其中包含SSH，（S）FTP，SMB / CIFS，DAAP媒体服务器，RSync，BitTorrent客户端等服务。
  • openSUSE (opens new window)- 一个开源的 Linux 发行版
  • OpenWrt (opens new window) -OpenWrt被描述为用于网络管理的嵌入式设备的Linux发行版。
  • OSMC (opens new window)- 基于Kodi的开源媒体中心。
  • PiDeck (opens new window)- 小型DVS系统，可让您使用黑胶唱片控制数字音乐文件。
  • Pi MusicBox (opens new window) - 在树莓派上播放音乐流
  • PiNet (opens new window)- 一个用于设置和管理教室里树莓派的集中式用户帐户和文件存储系统。
  • PirateBox (opens new window)- 匿名脱机移动文件共享和通信系统。
  • Plasma Bigscreen (opens new window) - 专供大屏幕电视的 Linux 系统
  • PliPlay (opens new window) - 为树莓派打造的开源视频游戏硬件
  • Puppy Linux (opens new window) - 微型的 Linux 发行版
  • RaspBSD (opens new window) - FreeBSD 的免费开源镜像版本
  • RaspiOS（原 Raspbian） (opens new window)- 官方支持的树莓派镜像，基于Debian并提供精简版。
  • RasPlex (opens new window) - 树莓派的Plex客户端。
  • Recalbox (opens new window)- 在树莓派上拖放轻型复古游戏和媒体中心。
  • RetroPie (opens new window)- 在树莓派上进行复古游戏。
  • RISC OS (opens new window)- 用于ARM设备的快速，轻松定制的操作系统。
  • Rocket Show (opens new window) - 在舞台上实时播放同步的背景音乐，视频和DMX序列。
  • RuneAudio (opens new window)- 免费和开源的OS，可将嵌入式硬件转变为Hi-Fi音乐播放器。
  • SamplerBox (opens new window)- 即插即用采样器：将.WAV样本拖放到SD卡上，然后播放！
  • SARPi (opens new window)- 代表树莓派上的Slackware ARM。
  • SliTaz ARM (opens new window) - 一个免费小巧的 GNU/Linux 发行版
  • SUSE Linux (opens new window) - 德国 SuSE Linux AG 公司发行维护的Linux发行版
  • TrueOS (opens new window)- 基于FreeBSD的瘦客户端解决方案。
  • Twister OS (opens new window) - Twister OS 的主要目标是为开箱即用的单板计算机提供真正的桌面计算体验。
  • Ubuntu Core (opens new window) - IoT的官方（最小）Ubuntu发行版。
  • Ubuntu Kylin (opens new window) - 优麒麟操作系统是 Ubuntu 官方衍生版，专为中国市场而设计
  • Ubuntu MATE (opens new window)- 基于MATE桌面的树莓派版Ubuntu发行版。
  • Ubuntu Server (opens new window) - 树莓派的Ubuntu 服务器发行版。
  • UOS (opens new window) - 美观易用、极简操作的国产 Linux（基于 Deepin）
  • Volumio (opens new window)- 无头发烧级音乐播放器，旨在以最高保真度播放音乐。
  • Weather Station (opens new window)- Raspberry Pi Oracle Weather Station 的 Raspbian 镜像
  • Windows 10 ARM (opens new window)- 完整的 Windows 10 ARM64 桌面版
  • Windows 10 IoT核心版 (opens new window)- IoT的Windows 10发行版。
  • XBian (opens new window)- 一个小型，快速，轻巧的媒体中心Linux发行版
